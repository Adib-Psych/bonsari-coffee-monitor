🚀 Setup Google Sheets Sync

Bonsari Coffee Entry Dashboard · Connect ke Google Sheets · ~15 menit total
📋 Apa yang akan kita lakukan: Buat Google Sheets sebagai database, deploy Apps Script sebagai webhook receiver, paste URL ke Entry Dashboard. Setelah setup, setiap submit di Entry Dashboard otomatis ter-sync ke Google Sheets — bisa diakses dari HP, share ke tim, dan tetap punya backup historical lengkap.
1
Buat Google Sheets baru
⏱ 1 menit
  1. ✅ Sudah ada! Buka Google Sheets yang sudah disiapkan: Bonsari_Coffee_Entry_Sync
  2. Tidak perlu setup struktur tab — Apps Script akan auto-create 5 tab (Sales_GB, Sales_RB, Pengeluaran, Sortasi, Adjustments, Logbook, Customers) saat sync pertama
  3. Pastikan login dengan akun yang punya akses (asroriadib@gmail.com atau bonsaricoffee@gmail.com)
💡 Sheets akan otomatis bikin 5 tab (Sales_GB, Sales_RB, Pengeluaran, Sortasi, Adjustments) saat entry pertama di-submit. Tidak perlu setup manual.
2
Buka Apps Script editor
⏱ 30 detik
  1. Di Google Sheets yang barusan dibuat, klik menu Extensions → Apps Script
  2. Tab baru akan terbuka dengan editor Apps Script
  3. Hapus semua code default yang ada (biasanya cuma function myFunction() {})
3
Paste Apps Script code
⏱ 1 menit

Klik tombol Copy Code di bawah, lalu paste ke Apps Script editor (Ctrl+V atau Cmd+V):


    
    
      
  1. Setelah paste, klik icon 💾 Save atau Ctrl+S
    2. 
      
  2. Beri nama project: Bonsari Entry Sync
    3. 
    

  







  

    
4

    
Deploy sebagai Web App

    
⏱ 2 menit

  

  

    
    
      
  1. Klik tombol Deploy (kanan atas) → New deployment
    2. 
      
  2. Klik icon ⚙️ (gear) di sebelah "Select type" → pilih Web app
    3. 
      
  3. Isi settings sebagai berikut:
        
      
          
    • Description: Bonsari Coffee Entry Sync v1
      • 
          
    • Execute as: Me (asroriadib@gmail.com)
      • 
          
    • Who has access: Anyone
      • 
        
    
      
    4. 
      
  4. Klik Deploy
    5. 
    

    

      ⚠️ Authorize permissions: Akan muncul popup minta authorize. Klik "Authorize access" → pilih akun Google Anda → klik "Advanced" → "Go to Bonsari Entry Sync (unsafe)" → "Allow". Ini muncul karena script ini buat Anda sendiri (belum verified Google), aman karena code-nya transparan dan Anda yang buat.
    

  







  

    
5

    
Copy Web App URL

    
⏱ 30 detik

  

  

    
    
      
  1. Setelah deploy berhasil, akan muncul popup dengan URL Web app
    2. 
      
  2. URL formatnya: https://script.google.com/macros/s/AKfycb.../exec
    3. 
      
  3. Klik Copy di sebelah URL
    4. 
      
  4. Klik Done untuk close popup
    5. 
    

    

      ✅ Save URL ini di tempat aman (note app, password manager, dll). Kalau hilang, perlu redeploy.
    

  







  

    
6

    
Paste URL ke Entry Dashboard

    
⏱ 30 detik

  

  

    
    
      
  1. Buka Entry_Dashboard_Bonsari.html
    2. 
      
  2. Di sidebar kiri, klik tab ⚙️ Settings & Sync
    3. 
      
  3. Paste Web App URL ke field "Webhook URL"
    4. 
      
  4. Klik di luar field (atau Tab) — akan ada toast "Webhook URL saved"
    5.
